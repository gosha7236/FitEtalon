Plugin to manage Workouts and leaderboards.

== Description ==

LeaderBoard plugin is to enhance a Wordpress site to manage events and LeaderBoards; could manage events, divisions,workouts, competitors and scoreboard associated with each events. All the admin forms are powered with Ajax.

You can find the PRO version of this plugin from <a href="https://wpleaderboard.com">https://wpleaderboard.com</a>

= Translations =

Only English version of the plugin is available right now.

== Installation ==

1. Upload the entire `LeaderBoard` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find LeaderBoard menus in your WordPress admin panel.
